The figure encapsulates self-contained content on a webpage like illustrations, diagrams, photos, code listings, etc. The figure element can contain an optional figcaption that is used as a caption or legend for the figure. The figcaption must be the first or the last child of the figure element.

Together the figure and figcaption elements can be used to annotate images.

Accessibility

  • The ARIA role="group" is used to indicate the semantics of the figure and the labelling relationship of figcaption to assistive technologies.
  • Until the figure is fully supported by all browsers you should provide a aria-labelledby (mandatory at ING) to ensure that the figcaption is seen as the label for the figure.
  • When the figure is referred to from the main content of the document, it enables the figure to be moved away but without affecting the flow of the document. That's why labels should be used to refer to figures, rather than relative references (ex.: "See figure no.3" instead of "See below").

Multiple images can be nested in a figure with a single figcaption

The figure itself can be nested to provide both a group caption and individual captions for each figure in the group.
