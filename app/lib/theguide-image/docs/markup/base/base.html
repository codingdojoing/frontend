The img tag has two required attributes: src and alt.

Accessibility

  • The alt attribute on images is a very important accessibility attribute. Authoring useful alt attribute content requires the author to carefully consider the context in which the image appears and the function that image may have in that context. Additional guidance and techniques are available in Resources on Alternative Text for Images.
  • When the image is used for decorative purposes only, make the alt text empty (alt=""). This indicates to assistive technologies that the image can be safely ignored. Do not add ARIA properties. Unnecessary use of ARIA roles, states and properties will have a negative impact on accessibility.
