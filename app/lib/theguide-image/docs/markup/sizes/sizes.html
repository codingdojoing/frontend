The size of images can be set by specifying the width of the image. The height is proportionally set by the aspect-ratio of the image.

When no size is specified the image will be displayed with the original sizes. The sizes are specified by component sizes and available in the following options:

  • xs
  • md
  • lg
  • xl
  • xxl
  • xxxl
  • jb
  • xj
  • xxj
  • xxxj
  • mg
  • xmg
  • xxmg
  • xxxmg
  • gg
  • xgg
  • xxgg
  • xxxgg

For larger sized images use img-responsive in combination with e.g. the grid system.

Aspect ratio

The aspect ratio of an image describes the proportional relationship between its width and its height. Use the same picture but cropped with a different aspect ratio to set the height of the image.
