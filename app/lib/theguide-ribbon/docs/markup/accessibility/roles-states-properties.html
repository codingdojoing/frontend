Lists

To prevent two separate lookup lists in assistive tools, the menu items ribbon-list and ribbon-dropdown-menu are defined as role presentation. A complete set of menu items must be accessible without same targeted links. Links that are doubles and not visible should have the aria-hidden attribute.
