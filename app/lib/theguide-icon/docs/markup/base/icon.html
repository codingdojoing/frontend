Incorporating icons takes a few steps:

  1. Place an <span> element (required!) with class icon.
  2. Add a specific icon by adding the icon name class: icon-*.
    List of single icon names
    List of stacked icon names
  3. Add the attribute aria-hidden="true" to the <span> element, to hide this element from assistive technologies. Unfortunately, VoiceOver on OS X does read the content of pseudo elements.

When placing icons before or after strings of text, as in buttons or (navigation) links, be sure to leave a space between the <span> element and the text for proper spacing.
